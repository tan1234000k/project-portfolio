## 💻 Code

This folder contains the embedded and interface code for the project.

### Arduino Code
- Reads LDR sensor values
- Controls dual-axis servo motors
- Measures voltage and current
- Computes power output
- Sends data via Bluetooth

### Web Interface
- Displays real-time voltage, current, and power graphs
- Built using HTML and JavaScript (Chart.js)

The system operates as a closed-loop feedback control system with real-time monitoring.
